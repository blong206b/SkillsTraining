Data Roles Decision Tree

Do you prefer working with raw data or analyzing data?

Data Engineer

As a Data Engineer, you will be responsible for collecting, storing, and processing data. This involves creating and maintaining data pipelines, databases, and data storage solutions.

You'll work closely with Data Scientists and Analysts, making the data accessible and usable for them. Knowledge in SQL, Python, and cloud services is often required.

Data Analyst

A Data Analyst interprets data to provide actionable insights for the company. They create dashboards, graphs, and visual presentations to help the business make informed decisions.

Typically, a Data Analyst will need to be proficient in statistical tools, SQL, and data visualization software like Tableau.
